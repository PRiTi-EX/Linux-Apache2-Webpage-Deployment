# Linux-Apache2-Webpage-Deployment
This project demonstrates how to deploy a simple webpage on an Apache2 server using Linux localhost.

The webpage was created using:
HTML
CSS
JavaScript

The webpage displays common Linux commands in a terminal-style interface.
Technologies Used:
Linux
Apache2
HTML5
CSS3
JavaScript
Apache2 Commands

####Commands###
Start Apache2:
sudo systemctl start apache2
Check Apache2 status:
sudo systemctl status apache2
Stop Apache2:
sudo systemctl stop apache2

###Project Setup###
Go to Apache web directory:
cd /var/www/html
Create HTML file:
nano priti.html
Open in browser:
http://localhost/priti.html

###Features###
Terminal-style Linux themed UI
Responsive webpage layout
Linux command reference table
Simple Apache2 deployment
